# FileSystem

### Estrutura dos Ficheiros

O código foi modularizado para separar a lógica principal das definições de dados e das funções auxiliares, estando dividido da seguinte forma:

* **`LeoMartelliMain.c`**: Ficheiro principal responsável por ditar o fluxo de execução do programa e realizar a chamada e gestão do sistema de ficheiros.
* **`Structs.h`**: Ficheiro de cabeçalho que contém exclusivamente as estruturas de dados. É aqui que estão definidas as bases para a implementação das listas e filas essenciais para o funcionamento do código.
* **`Functions.h`**: Ficheiro de cabeçalho que abrange todas as funções criadas para complementar o ficheiro principal, organizando a manipulação lógica das estruturas definidas anteriormente.
